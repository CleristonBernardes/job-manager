# Job Manager
Manage tradies' job assignment


## Execute the API
- node 8.4.0
```sh
$ npm install
$ npm start
```

## Test considerations

### Config
Verbose config
- Set verbose true to see the summary of each test (including performance), but not logs
- Set verbose false to see the log generated by each test.
```
"verbose": true
```

### Execute
```
npm test
```
or
```
npm t
```
